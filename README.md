# Dotfiles

## Installation
```sh
$ git clone git@github.com:jpxcz/dotfiles.git ~/dotfiles
```

## Dependencies
[GIT](https://git-scm.com/)  
[Tmux Plugin Manager](https://github.com/tmux-plugins/tpm)

## Folders
```sh 
$ ./folders.sh
```

## Config files
```sh
$ ./link.sh
```

## Arch Linux dependencies installation
```sh
$ sudo ./arch.sh -i
```
